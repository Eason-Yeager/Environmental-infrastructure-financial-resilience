## Notes
- All `.tex` files are ready for direct `\input{}` in the LaTeX manuscript.
- All `.pdf` figures use 300 DPI and are sized for two-column journal format (default 10×5 inches, adjustable in scripts).
- The `data/processed/` directory (not tracked by git) stores intermediate `.rds` files passed between scripts.
